The Big Question

What is the Fibonacci Sequence? ü§î

‚Ä¢ It's a series of numbers where the next number is found by adding up the two numbers before it.

‚Ä¢ It usually starts with 0 and 1.

Sequence:

0, 1, 1, 2, 3, 5, 8, 13, 21, ‚Ä¶

The Rule:

ncurrent = nprev1 + nprev2

Code Breakdown: The while Loop

The Engine (The while Loop):

while count < nterms:
    # ... code runs here ...
    count += 1  # IMPORTANT: moves us closer to the goal

‚Ä¢ A while loop repeats a block of code as long as the condition (count < nterms) is TRUE.

‚Ä¢ When count reaches nterms, the loop stops.

üìù Notes for Cover Teacher:

Explain that this is what keeps the process going. If we forget count += 1, the loop runs forever! (An infinite loop - a bad thing!)

Slide 6 of 8

Code Breakdown: The Magic!

Generating the Next Number:

nth = n1 + n2
n1 = n2
n2 = nth

Step 1: Calculate the Next Term (nth)

Add the two previous numbers together

Step 2: Shift 1 - The old n2 becomes the new n1

Move the second number to the first position

Step 3: Shift 2 - The new nth becomes the new n2

Move the new number to the second position

üìù Notes for Cover Teacher:

This is the most important slide. Use your hands to demonstrate the 'shift' of the two previous numbers.

Example: Start: n1=3, n2=5. ‚Üí 1. nth is 8. ‚Üí 2. n1 becomes 5. ‚Üí 3. n2 becomes 8.
